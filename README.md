# Meeting App

This project is a meeting application developed with ASP.NET Core MVC.

## Project Features
- MVC project structure
- Controllers, Models, and Views
- Layout and page design
- Form operations
- Model binding
- Tag Helpers
- Validation operations
- Participant listing and detail pages

## Technologies Used
- C#
- ASP.NET Core MVC
- HTML
- CSS
- Libman

## Purpose
The purpose of this project is to improve ASP.NET Core MVC development skills and build a basic meeting management application.

## Note
This project was developed for educational purposes.
